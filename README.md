# Dns Tunnel Detector

DNS tunneling detection tool.

Alat deteksi DNS tunneling.


## Description

Analyzes DNS traffic patterns to identify potential tunneling for data exfiltration.

## Deskripsi

Menganalisis pola lalu lintas DNS untuk mengidentifikasi potensi tunneling untuk eksfiltrasi data.


## Installation

- Navigate to the project folder
- No dependencies required

## Instalasi

- Masuk ke folder proyek
- Tidak perlu dependensi


## Usage

- Analyze log: python dns_tunnel.py /var/log/dns.log
- Monitor interface: python dns_tunnel.py --interface eth0
- Demo: python dns_tunnel.py --demo

## Penggunaan

- Analisis log: python dns_tunnel.py /var/log/dns.log
- Monitor interface: python dns_tunnel.py --interface eth0
- Demo: python dns_tunnel.py --demo
